# laobi

Use Django Framework

## Prerequisite
- python

In Ubuntu 18.04


Install pip
```
sudo apt install python3-pip
```

Install venv
```
sudo apt install python3-venv
```

Create virtual environment
```
python3 -m venv my_env
```

Activate my_env
```
source my_env/bin/activate
```

Install django
```
pip install django
```
